RICO POLLO - PROTOTIPO LOCAL

Instrucciones rápidas:

1) Coloca este proyecto en tu htdocs de XAMPP (ya estás en c:/xampp/htdocs/ricopollo)

2) Ejecuta el setup para crear la base de datos y el usuario `SCARY` (contraseña por defecto `SCARY1234`):

```bash
php setup.php
```

3) Abre el panel de administración en el navegador apuntando a `public/login.php`.

4) Credenciales por defecto:
- Email: SCARY@LOCALHOST
- Contraseña: SCARY1234

NOTAS:
- Todos los textos y cadenas se muestran y guardan en MAYÚSCULAS.
- Cambia la contraseña del usuario `SCARY` después del primer inicio.
- Este es un prototipo mínimo para autenticación; integraremos más funciones (productos, pedidos, impresión) en próximos pasos.
